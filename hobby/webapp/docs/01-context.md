## Success Criteria

**We'll know it's working when:**
- [ ] 10+ active users
- [ ] Users complete 5+ tasks per week
- [ ] Page loads in < 2 seconds
- [ ] Works on mobile browsers

---

## Constraints

**Technical:**
- Free hosting (Vercel + Supabase)
- Single developer
- Must work on all devices

**Time:**
- MVP in 2 weeks
- 2-3 hours per day development

**Budget:**
- $0-10 per month max
- Use free tiers only

---

## Out of Scope (For Now)

**Not building:**
- Native mobile apps
- Team collaboration features
- Advanced analytics
- Payment/subscriptions
- API for third parties
- Offline mode

**Keep it simple!**

---

## Tech Stack

**Frontend:**
- React + Vite (or Next.js)
- Tailwind CSS

**Backend:**
- Supabase (PostgreSQL + Auth)
- Or: Node.js/Python if needed

**Hosting:**
- Vercel (frontend) - free
- Supabase (database) - free tier
