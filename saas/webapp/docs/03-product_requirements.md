# 03. Product Requirements (PRD)

## 3.1 Functional Requirements (User Stories)

### Authentication Module
- **US-1**: As a user, I want to sign up with email/password so I can access the platform.
- **US-2**: As a user, I want to reset my password via email link.

### Dashboard Module
- **US-3**: As a user, I want to see a summary graph of my sales.
- **US-4**: As a user, I want to filter data by date range (Last 7 days, 30 days, Custom).

## 3.2 Non-Functional Requirements
- **Performance**: Dashboard loads in < 2 seconds.
- **Scalability**: Support 1000 concurrent users.
- **Security**: Data encrypted at rest; Rate limiting on public APIs.
- **Compatibility**: Chrome, Firefox, Safari, Edge (Latest 2 versions).
