# 06. Store & Release

## 6.1 App Store Guidelines
- **Apple**: Review Human Interface Guidelines. Check Privacy Nutrition Label.
- **Google**: Adhere to Play Store Policy.

## 6.2 Signing & Secrets
- **Keystores**: `upload-keystore.jks` and `ios_distribution.p12`.
- **Storage**: Store encrypted in secure vault/CI secrets. Never in repo.

## 6.3 Release Tracks
- **Alpha**: Internal QA (App Distribution).
- **Beta**: External Testers (TestFlight / Play Console Open Testing).
- **Production**: Public Release.
