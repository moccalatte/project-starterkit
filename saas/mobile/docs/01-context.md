# 01. Mobile App Context

## 1.1 Executive Summary
*Mobile Application for iOS and Android.*
> Example: A companion app for field technicians to view tasks and report status offline.

## 1.2 Business Objectives
1.  **Efficiency**: Reduce reporting time by 50%.
2.  **Adoption**: 100% of field technicians using app within 1 month.
3.  **Reliability**: 0 data loss during offline sync.

## 1.4 Scope
### In Scope
- Task listing.
- Photo upload.
- Offline mode (local database).

### Out of Scope
- User registration (Account created by Admin on Web).

## 1.5 Constraints
- **Device Support**: Android 10+, iOS 15+.
- **Network**: Must work in low-connectivity areas (Edge/3G).
