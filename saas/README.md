# SaaS Project Templates

This directory contains starter kits and documentation for various types of software projects.

## Available Projects

### [Web Application](webapp/)
Full-stack web application template.
- **Frontend**: React/Next.js
- **Backend**: Node/Python/Go
- **Key Docs**: Architecture, API, Security

### [Bot](bot/)
Template for conversational agents (Telegram, Discord, Slack).
- **Key Docs**: Commands, Conversational Flows, State Management

### [Mobile App](mobile/)
Template for iOS/Android applications.
- **Key Docs**: App Concept, Store Release, Offline Sync

### [AI/ML](ai/)
Template for Machine Learning and AI projects.
- **Key Docs**: Data Pipeline, Model Architecture, Evaluation

## Getting Started
Navigate to the specific folder for your project type and follow the `README.md` there.
